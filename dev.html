Development Tools and Notes

Sublime Tips

Sublime Text is an awesome text editor. It has a vibrant community with a large library of useful and lightweight plug ins. For most of these you can substitute CTRL for CMD if you are using Windows.

Word/Line Selection

To select a word, or once a word is selected to select others instances. 
CMD+D
To select a line. 
CMD+L
To select current scope. 
CTRL+CMD+M

Sorting

Plain old F5. Simple and effective. Good to use with CSS.

Command Menu

Still learning everything sublime text can do here, if the package manager is installed you can access it via this tool. You can also change syntax highlighting options quickly from here. 
CMD+SHIFT+P
Also useful is the fuzzy search using the below 
CMD+P
You can also type : to jump to line numbers and @ to jump to modules or functions in code. Or you can get rid of the distraction 
CMD+SHIFT+CTRL+F

Navigation

If you want to jump between tabs use 
CMD+T
To navigate left/right through tabs 
CMD+SHIFT+[
CMD+SHIFT+]
Find Window 
CMD+SHIFT+F
Populate selected word into the find window. 
CMD+E
REALLY HANDY, use this to navigate through modules/functions. Useful especially with CSS. 
CMD+R
Side by side docs For Horizontal 
CMD+ALT+2-4
For Vertical 
CMD+SHIFT+ALT+2-4
