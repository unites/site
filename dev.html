Sublime Text

Sublime Text is a personal favorite of mine. It has a vibrant community with a large library of useful and lightweight plug ins. These shortcuts are for OSX, most of these you can substitute Control for Command if you are using Windows.

Most used Sublime Hot Keys

Select Line Command + L
Select Current Scope Control + Command + M

Emmet

Emmet is a toolkit for web development. It allows you to use shorthand to write full HTML and CSS. I use it with Sublime Text and VIM but it can be used with many other text editors. For sublime text all you need do is hit tab after any of the shorthand you type. (Note, your file extension must be the supported type, i.e. html/css).
